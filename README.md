## 🚀 Getting Started

* **Prerequisites**: Install the Flutter SDK and platform tooling for Android/iOS. Confirm your setup by ensuring `flutter doctor` runs clean.
* **Open a Project**: Navigate to the specific project directory using `cd <project-folder>`, then run `flutter pub get`.
* **Run on Device/Emulator**: Execute `flutter run -d <device-id>`.
* **Run Tests**: Execute `flutter test` from the project root.

---

## 📝 Per-Project Notes

* **Dependencies**: Each project lists its specific dependencies in its own `pubspec.yaml` (always remember to run `flutter pub get`).
* **Platform Files**: Folders for Android, iOS, Web, macOS, Windows, and Linux may be present depending on the project. Simply select the platform you are targeting.

---

## 🤝 Contributing / Usage

* **Explore**: Open any project folder in VS Code or Android Studio to inspect the code and run it directly.
* **Reuse**: Feel free to copy smaller examples into new projects to experiment safely.
