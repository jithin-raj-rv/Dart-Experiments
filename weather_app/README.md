# weather_app

## Description

A Flutter weather application that fetches real-time weather data using HTTP requests. It uses GetX for state management, Lottie for animated weather icons, and Geolocator/Geocoding for location-based weather lookups.

## Setup

```bash
flutter pub get
```

## Run

```bash
flutter run
```

**Note:** You will need to configure your own API key for weather data in the source code before running.
